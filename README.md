# Godot_Pong
 A Classic Pong clone made with Godot 4.6. My first completed game project.

## Implemented
- Two Paddles
- ball
- scoreboard
- screen boundaries
  
## KEYs
- **SPACE** (Start: Throw ball)
- **ESC** (Restart: Reset scores)
- **W** (Left Paddle Up Movement)
- **S** (Left Paddle Down Movement)
- **UP_ARROW** (Right Paddle Up Movement)
- **DOWN_ARROW** (Left Paddle Down Movement)
